
Deterministic Finite Automaton
==============================

Definition:
-----------

    M : (Q, Σ, δ, q0, F)

    Q                   finite set of internal states
    Σ                   finite set of symbols -- input alphabet
    δ : Q x Σ -> Q      total function -- transition function
    q0 ∈ Q              initial state
    F ⊆ Q               set of final states
